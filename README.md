# AI课程设计助手 - 职业教育版

## 项目信息

- **项目名称**：AI课程设计助手
- **版本号**：v1.0.0
- **开发周期**：14天（2025-01-XX ~ 2025-01-XX）
- **技术栈**：Electron + React + SQLite + Deepseek API + 导出与模板引擎

## 项目简介

面向职业院校教师的AI驱动课程设计工具，借鉴NotebookLM的笔记本理念，提供从需求探索到资源包导出的完整工作流程。

## 核心功能

### P0 核心功能
- 创建课程笔记本
- AI生成教学设计框架
- 逐模块编辑（6个模块）
- AI生成模块详细内容
- 教学过程表格编辑
- 资源上传和管理
- 导出Word教学设计（25页）
- 导出PPT课件（3种模板，50-80页）

### P1 高级功能
- AI生成图片（豆包/文心/通义）
- 三层资源验证（元数据、相似性、盲点诊断）
- Word批注协作（识别、应用、连锁更新）
- 模板复用系统（保存、创建、模块复用）
- 版本管理
- 导出完整资源包
- 课程信息图生成

## 工作流与技术架构

完整流程图、模块拆分与IPC设计请见 `整体的产品工具介绍.md`。

## 技术架构

```text
Electron主进程
├── 数据库管理（SQLite）
├── API调用（Deepseek）
├── 文件操作（资源管理）
└── 导出功能（Word、PPT、ZIP）

React渲染进程
├── 笔记本管理
├── 教学进度表编辑
├── 教学框架版本管理
└── 生成与导出配置
```

## 开发规范

### 目录说明

- `docs/` - 所有文档
- `src/` - 源代码
  - `src/main/` - Electron主进程代码
  - `src/renderer/` - React前端代码
  - `src/preload/` - 预加载脚本
- `prompts/` - AI Prompt模板
- `templates/` - Word/PPT模板配置
- `resources/` - 静态资源
- `tests/` - 测试文件
- `scripts/` - 构建脚本
- `dist/` - 打包输出

## 安装与运行

### 开发环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0
- Windows 10/11

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 打包

```bash
npm run build
```
